# GameKey

Telegram bot for game key distribution and store management.

## Installation

```bash
# Install dependencies
npm install

# Development
npm run dev

# Build
npm run build

# Production
npm run start:prod
```

## Features

- Game key distribution via Telegram
- Payment processing
- Store management

## Technologies

- Node.js
- TypeScript
- Grammy (Telegram Bot API)
- MongoDB
- Express
